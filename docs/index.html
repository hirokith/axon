Deep Observability

💬

Rich IDE-like Chat Interface

Multi-session with persistent history. Full Markdown and LaTeX math rendering. Watch tokens stream in real-time alongside a live thinking display. Each session gets its own isolated working directory.

📡

RAW Protocol Mode

See every JSON-RPC message on the wire. No need to open Wireshark. Inspect the exact payloads being sent and received over the stdio transport.

🔌

MCP Server Management

Configure Model Context Protocol servers natively. Attach MCP servers (stdio or HTTP) to individual sessions. Supports full CRUD and JSON bulk import.

🛠️

Structured Tool Calls & Outputs

Stop guessing what tools your agent is calling. Expandable tool call cards with syntax-highlighted inputs/outputs, sequential numbering, execution duration, and file preview modes.

ACP Protocol Support

Axon implements the Agent Client Protocol fluently, acting as the perfect client for any ACP-compatible agent.

  • Handshake initialize
  • Context Setup session/new w/ CWD
  • Execution session/prompt
  • Streaming session/update
  • Security Permission dialogs
